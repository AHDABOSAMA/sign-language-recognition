# Sign Language Recognition ✋🤟

This project uses deep learning and computer vision to recognize English sign language alphabets in real time.

## Features
- Real-time hand gesture recognition
- CNN model trained on 87,000+ images
- Built using TensorFlow and OpenCV

## Tech Stack
- Python
- TensorFlow
- OpenCV
- NumPy

## Files
- model.hdf5 → trained model
- Python script → runs detection

## How to Run
1. Install requirements:
   pip install -r requirements.txt

2. Run the program:
   python your_script_name.py
